Как запустить проект на другом ПК
==================================

Рабочее React-приложение лежит в папке app/ (папка node_modules
специально не копировалась — она платформозависимая, её надо
поставить заново на целевом компьютере).

1. Установи Node.js (проверено на v24, подойдёт любой current LTS):
   https://nodejs.org/

2. В терминале:
     cd printfee-project/app
     npm install
     npm run dev

3. Открой ссылку из терминала (обычно http://localhost:5173/).

Для сборки продакшен-версии вместо "npm run dev":
     npm run build
     npm run preview

---

Остальные файлы в корне проекта (index.html, desktop.html,
constructor.html и т.д.) — старый статичный HTML-прототип
(дизайн-хендофф), не рабочий сайт. Его можно открыть в браузере
напрямую двойным кликом, npm/установка для него не нужны.
