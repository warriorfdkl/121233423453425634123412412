Printfee — дизайн-проект

Печать одежды с собственным дизайном (DTF, Казань). Три готовых экрана + рабочий прототип конструктора принта. Ниже — точки входа для просмотра.

Экраны
Мобильная главная
Hero, статы, каталог, отзывы, FAQ. Открывается в iOS-фрейме 430×880.
Mobile
Десктопная главная
Canvas-волна в hero, showcase, sticky news-стек.
Desktop
Конструктор принта
Рабочий прототип: загрузка фото, текст, drag/rotate/resize, цвет, размер, корзина.
Interactive
Каталог готовых дизайнов
Сетка готовых футболок в духе brandshop.ru: фильтры, сортировка, избранное — с рабочим клиентским фильтром по категориям.
Interactive
Избранное
Сохранённые дизайны с рабочим удалением из списка и пустым состоянием.
Interactive
Корзина
Позиции с количеством, пересчётом суммы и доставки в реальном времени, пустое состояние.
Interactive
Личный кабинет
Профиль, история заказов, личные данные и адреса — с переключением вкладок.
Interactive
Черновики / эксплорейшн (для контекста решений)
